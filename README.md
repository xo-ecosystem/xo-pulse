# XO Pulse

Where signals become artifacts.

Pulse publishes entries that can be referenced later — signed or unsigned, simple or consequential.
If nothing is listed here yet, that is expected.

Publication flow:
- Write → sign (optional)
- Publish → durable output
- Summarized daily in Digest

Signed → Visible → Alive

- XO Center: https://xo.center
- Daily Digest: https://xo-digest.com
- Verification (Vault): https://xo-vault.com

Pulse is a surface. The record lives elsewhere.
